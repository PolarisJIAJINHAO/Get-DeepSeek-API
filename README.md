# Get-DeepSeek-API
1. 在DeepSeek官网注册开发者账号，获取调用API资格。需要给DeepSeek充值，但很便宜，我过往的经验是，对于简单任务10块钱应该能用于15000多条数据。你应该已经使用关键词初筛过一遍了，我感觉10块钱应该能完成你的任务。
2. 创建API Key, 复制并保存好你的API，别告诉任何人。
3. 将代码复制到你的python解释器中。
4. 打开终端，输入：setx DEEPSEEK_API_KEY "你的API key"。引号里输入你刚才保存好的API Key。
5. 在终端中安装所有依赖：pip install -U openai pandas openpyxl tqdm
6. 将代码中把你需要改的所有基础配置改成你的文件格式。
7. 更改代码的分类提示词，让提示词更符合你的任务，这是个例子，你可以自己优化一下：
     “任务：你是一个数据分析助手。请判断以下微博评论是否与“铁路购票体验”相关。
    【判断标准】：
    - 相关：包括买票难易、12306系统崩溃/卡顿、候补票、抢票加速包、票价贵贱、退改签规则、黄牛问题等。
    - 不相关：包括单纯的车站/列车风景打卡、车厢内发生的趣事（与买票无关）、无关的广告、单纯提及“坐火车”但没提购票过程。
    【输出要求】：
    请仅输出一个字：“是” 或 “否”。不要包含任何解释或其他标点。”
8. 将代码中对应的元素进行修改，如果你有一定的代码基础很好改，把自己的配置改进去就行。如果你代码写的不好，可以先把代码上传给AI，在每一次报错后，将错误代码复制粘贴过去，AI会提供解决方案，直接照着AI的方式改即可。这一部分相对麻烦一点，但几十分钟肯定能搞好。
9. 运行即可
